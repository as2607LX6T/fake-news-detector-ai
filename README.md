🧠 TruthLens AI — Fake News Detection Platform

An AI-powered fake news detection platform built using Machine Learning, NLP, FastAPI, and React.
Designed to identify misleading news articles in real time with high accuracy and an immersive modern UI.

🚀 Overview

TruthLens AI is a full-stack AI application that detects whether a news article is REAL or FAKE using Natural Language Processing and Machine Learning.

The system combines:

advanced text preprocessing
TF-IDF vectorization
multiple ML classifiers
FastAPI backend APIs
modern React frontend
real-time prediction workflows

The project was designed as a portfolio-grade AI system demonstrating:

ML engineering
backend API development
frontend integration
full-stack deployment
scalable project architecture
